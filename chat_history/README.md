# Chat History

One markdown file per session. Filename: `YYYY-MM-DD_NNN.md` where NNN is a sequence number if multiple sessions happen on the same day.

## Convention

At the end of each Claude Code session, Claude writes a session summary to this folder. The summary covers:
- What was discussed or decided
- What was built or changed
- Key decisions made (with brief reasoning)
- What to pick up next session

## Why this exists

Claude Code sessions don't persist conversation history between runs. This folder bridges the gap — it gives the next session enough context to resume without re-deriving everything from scratch.
