## STEP 2 — APPROVED

**What was built:**

`_serialise(obj: Any) -> Any`:
- Handles: `datetime` (BEFORE `date` — subclass ordering), `date`, `time`
  (PostgreSQL time columns from psycopg), `Decimal`, nested `dict`, `list/tuple`
- `datetime` checked before `date` is non-negotiable: `datetime` is a subclass of
  `date`; if `date` were first, datetime objects would lose their time component

`dispatch(tool_name, tool_input, run_id, step_index) -> tuple[Any, bool]`:

Three observation shapes (is_error semantics):
- **(a) success** — normal return value, `_serialise(result)`, `is_error=False`
- **(b) business-empty or signal** — `None`, empty list, shortfall dict — `is_error=False`
- **(c) system error** — `{"error": True, "type": ..., "message": ...}`, `is_error=True`

`is_error=True` ONLY on shape (c). `None` from get_absence is shape (b). Shortfall
from check_weekly_moq is shape (b). Only Python exceptions are shape (c).

**add_note special case in dispatch:**
- `dispatch()` calls `log_agent_step` internally: `tool_name=label` (not "add_note")
- Returns `(None, False)`
- Outer loop MUST skip its own `log_agent_step` call when `tool_name == "add_note"`
- `step_index` is owned by outer loop; incremented once per `tool_use` block

**CORRECTION LOCKED — `log_agent_step` real parameter:**
- Real signature uses `reasoning: str | None` — NOT `tool_output_summary`
- Design doc (session 013) had wrong parameter name

## STEP 3 — NEXT (not started)

**What to build:** The core `run()` loop replacing the `NotImplementedError` stub.

**Full spec (locked in session 013 + corrections from this session):**

```
run(trigger_reason: str) -> None

1. run_id = create_agent_run(trigger_reason=trigger_reason)
2. system_prompt = _load_system_prompt()
3. messages = [{"role": "user", "content": trigger_reason}]
4. step_index = 0
5. call_count = 0
6. MAX = settings.max_tool_calls_per_run  # 15

LOOP:
  response = client.messages.create(
      model=settings.agent_model,           # "claude-sonnet-4-6"
      max_tokens=4096,                      # or from settings if added
      system=system_prompt,
      tools=TOOL_SCHEMAS,
      messages=messages,
  )

  tool_use_blocks = [b for b in response.content if b.type == "tool_use"]

  if not tool_use_blocks:
      # Model finished (stop_reason == "end_turn" or no tools called)
      text = next((b.text for b in response.content if b.type == "text"), "")
      complete_agent_run(run_id, text)
      return

  tool_results = []

  for block in tool_use_blocks:
      tool_name = block.name
      tool_input = block.input  # dict

      # HARD CAP CHECK — before dispatch
      if call_count >= MAX:
          log_agent_step(
              run_id=run_id,
              step_index=step_index,
              tool_name="max_calls_reached",
              tool_input={"tool_name": tool_name, "call_count": call_count, "max": MAX},
              tool_output_full=None,
              reasoning=f"Hard cap {MAX} reached; aborting remaining {len(tool_use_blocks)} tool(s)",
              urgency="urgent",
          )
          step_index += 1
          tool_results.append({
              "type": "tool_result",
              "tool_use_id": block.id,
              "content": json.dumps({"error": True, "type": "HardCap",
                                     "message": f"Max tool calls ({MAX}) reached"}),
              "is_error": True,
          })
          continue  # NOT break — feed is_error to ALL remaining blocks

      # Dispatch
      result, is_error = dispatch(tool_name, tool_input, run_id, step_index)

      call_count += 1

      # Log step — SKIP for add_note (dispatch already logged it internally)
      if tool_name != "add_note":
          log_agent_step(
              run_id=run_id,
              step_index=step_index,
              tool_name=tool_name,
              tool_input=tool_input,
              tool_output_full=result if isinstance(result, dict) else {"value": result},
              reasoning=None,
              urgency="urgent" if is_error else "none",
          )
          step_index += 1

      # SAFETY RECORDS RULE — non-negotiable, after every compose_session_order
      if tool_name == "compose_session_order" and not is_error and isinstance(result, dict):
          for rec in result.get("safety_records", []):
              if rec.get("safe") is False:
                  log_agent_step(
                      run_id=run_id,
                      step_index=step_index,
                      tool_name="dietary_safety_violation",
                      tool_input={"enrolment_id": rec["enrolment_id"],
                                  "menu_item_id": rec["menu_item_id"]},
                      tool_output_full=rec,
                      reasoning=(
                          f"UNSAFE MATCH: {rec['student_name']} assigned "
                          f"{rec['meal_name']} but student tags "
                          f"{rec['dietary_tags']} not covered by meal tags"
                      ),
                      urgency="urgent",
                  )
                  step_index += 1
              if rec.get("other_allergy_notes"):
                  log_agent_step(
                      run_id=run_id,
                      step_index=step_index,
                      tool_name="unverified_allergy_note",
                      tool_input={"enrolment_id": rec["enrolment_id"]},
                      tool_output_full={"other_allergy_notes": rec["other_allergy_notes"],
                                        "student_name": rec["student_name"]},
                      reasoning=(
                          f"Unverified allergy note for {rec['student_name']}: "
                          f"{rec['other_allergy_notes']} — requires human verification"
                      ),
                      urgency="urgent",
                  )
                  step_index += 1

      tool_results.append({
          "type": "tool_result",
          "tool_use_id": block.id,
          "content": json.dumps(result),
          "is_error": is_error,
      })

  # All blocks processed — append assistant turn + tool_results to messages
  messages.append({"role": "assistant", "content": response.content})
  messages.append({"role": "user", "content": tool_results})

  # If cap was hit, make one UNCOUNTED final summarising call then exit
  if call_count >= MAX and any(r.get("is_error") for r in tool_results):
      final = client.messages.create(
          model=settings.agent_model,
          max_tokens=1024,
          system=system_prompt,
          tools=TOOL_SCHEMAS,
          messages=messages,
      )
      text = next((b.text for b in final.content if b.type == "text"), "")
      complete_agent_run(run_id, f"[CAP HIT at {call_count}/{MAX}] {text}")
      return

  # Continue loop

# Exception handler (wraps entire loop):
except Exception as e:
    complete_agent_run(run_id, f"[EXCEPTION: {type(e).__name__}] {e}")
    raise
```
