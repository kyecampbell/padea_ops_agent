## Key decisions made

| Decision | What was decided | Why |
|---|---|---|
| V4/V5 cancelled | Build V3 as-is | Time constraint; V3 is already well-designed and complete. V4/V5 were simplification passes that would delay the build. |
| psycopg v3 | Use `psycopg[binary]>=3.1.0`, not `psycopg2-binary` | psycopg v3 is the modern, async-capable version. Aligns with Python 3.12 target. |
| agent_context/ pattern | Runtime agent reads from `agent_context/`, not `docs/` | Docs are design artefacts; agent_context/ is operationally shaped. Compact, runnable, no design noise. |
| email_mode: demo | Default is demo, all emails redirect to `kyec898@gmail.com` | Prevents accidental production sends during dev. Must be flipped manually for live demo only. |
| uv as package manager | `uv venv --python 3.12` everywhere | System Python 3.13/3.14 has a broken pyexpat module on this machine. uv pins cleanly to 3.12. |
